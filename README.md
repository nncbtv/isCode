# Appunti Ingegneria del Software

Questa repository raccoglie il codice prodotto, i riassunti e gli schemi UML elaborati durante il corso di **Ingegneria del Software**, con particolare focus sulle **implementazioni dei pattern** in Java.

## A chi è utile?

Questo materiale è pensato per gli studenti che dovranno affrontare l'esame di Ingegneria del Software e vogliono un punto di partenza concreto per **comprendere ogni design pattern**, senza dover scrivere tutto da zero.
